API Documentation

Base URL

http://localhost:3000

1. Authentication Routes

POST /auth/login

Description: Authenticates a user and returns a JWT token.

Request:


{
  "username": "string",
  "password": "string"
}

Response:


{
  "token": "string"
}
            

{
  "error": "Invalid credentials"
}

2. Categories Routes

GET /categories

Description: Retrieves a list of all categories.

Response:


[
  {
    "id": "integer",
    "name": "string"
  }
]

GET /categories/:categoryId

Description: Retrieves a specific category by its ID.

Request:


{
  "categoryId": "integer"
}

Response:


{
  "id": "integer",
  "name": "string"
}
        

{
  "error": "Category not found"
}

POST /categories

Description: Creates a new category (requires admin authentication).

Request:


{
  "name": "string"
}

Response:


{
  "id": "integer"
}
            

{
  "error": "Required field name is missing"
}

PUT /categories/:categoryId

Description: Updates an existing category by its ID (requires admin authentication).

Request:


{
  "name": "string"
}

Response:


{
  "id": "integer"
}
            

{
  "error": "Required field name is missing"
}

DELETE /categories/:categoryId

Description: Deletes a category by its ID (requires admin authentication).

Request:


{
  "categoryId": "integer"
}

Response:


{
  "message": "Category deleted successfully"
}
            

{
  "error": "Category not found"
}

3. Products Routes

GET /products

Description: Retrieves a list of all products.

Response:


[
  {
    "id": "integer",
    "title": "string",
    "price": "number",
    "description": "string",
    "category_id": "integer",
    "image": "string (URL)"
  }
]

GET /products/:productId

Description: Retrieves a specific product by its ID.

Request:


{
  "productId": "integer"
}

Response:


{
  "id": "integer",
  "title": "string",
  "price": "number",
  "description": "string",
  "category_id": "integer",
  "image": "string (URL)"
}
        

{
  "error": "Product not found"
}

POST /products

Description: Creates a new product (requires admin authentication). Supports file upload for the product image.

Request:


{
  "title": "string",
  "price": "number",
  "description": "string",
  "category_id": "integer"
}

Form Data:


File field: image (optional)

Response:


{
  "message": "Product successfully created.",
  "product": {
    "id": "integer",
    "title": "string",
    "price": "number",
    "description": "string",
    "category_id": "integer",
    "image": "string (URL)"
  }
}
            

{
  "error": "All fields are required"
}

PUT /products/:productId

Description: Updates an existing product by its ID (requires admin authentication). Supports file upload for the product image.

Request:


{
  "title": "string",
  "price": "number",
  "description": "string",
  "category_id": "integer"
}

Form Data:


File field: image (optional)

Response:


{
  "message": "Product successfully updated.",
  "product": {
    "id": "integer",
    "title": "string",
    "price": "number",
    "description": "string",
    "category_id": "integer",
    "image": "string (URL)"
  }
}
            

{
  "error": "All fields are required"
}

DELETE /products/:productId

Description: Deletes a product by its ID (requires admin authentication).

Request:


{
  "productId": "integer"
}

Response:


{
  "message": "Product deleted successfully"
}
            

{

5. Cart Routes

GET /carts

Description: Retrieves the cart for the authenticated user.

Response:


{
  "id": "integer",
  "user_id": "integer"
}
        

{
  "error": "Cart not found!"
}

6. Purchases Routes

POST /purchases/:productId

Description: Purchases a specific product by its ID (requires authentication).

Request:


{
  "quantity": "integer"
}

Response:


{
  "message": "Thank you for purchasing. Please check the products before proceeding.",
  "data": {
    "product title": "string",
    "quantity": "integer",
    "total price": "number"
  }
}
            

{
  "error": "Requested quantity not available",
  "availableQuantity": "integer"
}

POST /purchases

Description: Purchases all products in the user's cart (requires authentication).

Response:


{
  "message": "Thank you for purchasing. Please check the products before proceeding.",
  "data": [
    {
      "product title": "string",
      "quantity": "integer",
      "total price": "number"
    }
  ]
}
            

{
  "error": "Cart not found or no products in cart"
}
            

{
  "error": "Requested quantity not available",
  "availableQuantity": "integer"
}
